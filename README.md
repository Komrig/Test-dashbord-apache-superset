#  Тестовый дашборд в apache-superset
Описываю как делал визуальный дашборд в apache superset на примере готового csv датасета по продаже шоколада.
## Про саму платформу apache-superset и где устанавливал
Является готовой опенсорс решением для визуализации данных в виде всяких графиков.\
Устанавливал на на заранее арендованный VPS от beget. Использую самописный сертификат шифрования, https используется, но браузер все равно будет ругаться.
## Про датасет
На сайте kaggle для теста всял csv с продажей шоколада от разных компаний в различных странах. \

## Подключение к apache superset
Чтобы сунуть датасет, нужно для этого подключить саму базу данных.\
Для данного проекта была выбрана sqlite, ибо проста в установке и подходит для небольших таблиц. Но в идеале можно и postgresql установить.

В самом терминале vps устанавливаем sqlite3 и создаем пустую таблицу. Далее копируем полный путь и в apache выбираем в настройках database connections > +database и в этом окне выбираем sqlite и вставляем ссылку.\
<img width="1920" height="950" alt="image" src="https://github.com/user-attachments/assets/ae6ee0e8-deb6-4cb3-b129-1534ef5eb278" /> \
В настройках нового подключения даем разрешение на добавление csv файлов.
Ну и в итоге просто закидываем csv файл, базу выбираем sqlite.
