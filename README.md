#  Тестовый дашборд в apache-superset
Описываю как делал визуальный дашборд в apache superset на примере готового csv датасета по продаже шоколада.
## Про саму платформу apache-superset и где устанавливал
Является готовой опенсорсной системой для визуализации данных в виде всяких графиков.\
Устанавливал на заранее арендованный VPS от beget.
## Про датасет
На сайте kaggle для теста взял csv с продажей шоколада от разных компаний в различных странах. \
Пример нескольких строк. \
<img width="467" height="215" alt="image" src="https://github.com/user-attachments/assets/8c466d8b-770e-4189-82c4-4934d7d08998" />

## Подключение к apache superset
Чтобы загрузить датасет, нужно для этого подключить саму базу данных.\
Для данного проекта была выбрана sqlite, так как она проста в установке и подходит для небольших таблиц. Но в идеале можно и PostgreSQL установить.

В самом терминале VPS устанавливаем sqlite3 и создаем пустую таблицу. Далее копируем полный путь и в apache выбираем в настройках database connections > +database и в этом окне выбираем sqlite и вставляем ссылку.\
<img width="1920" height="950" alt="image" src="https://github.com/user-attachments/assets/ae6ee0e8-deb6-4cb3-b129-1534ef5eb278" /> \
В настройках нового подключения даем разрешение на добавление csv файлов.
Ну и в итоге просто закидываем csv файл, базу выбираем sqlite.

## Создание чартов
Создаем чарт, выбираем наш датасет и сам визуальную модель table и нажимаем create new chart\

Для этого в query mode ставим raw records и вставляем в columns все наши столбцы, за исключением amount, потому что это текстовое поле с суммой продаж, до этого мы перевели этот столбец в числовой формат\
